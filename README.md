# Mobile-Sales-Dashboard
Interactive Power BI dashboard analyzing mobile sales performance with KPIs, yearly/quarterly/monthly trends, and Same Period Last Year (SPLY) comparison. Built using DAX, Power Query, and interactive slicers for model, payment method, and time.
